# Projeto de gerenciador de salas de reuniões com Java e Angular


### OBJETIVOS
O objetivo desse projeto é desenvolver uma API com Spring Boot para gerenciar salas de reuniões.

### TECNOLOGIAS
* lombok
* H2 Database 
* Spring Web e Spring Data JPA
